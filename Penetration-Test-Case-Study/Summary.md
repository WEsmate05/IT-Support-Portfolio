# Penetration Testing Case Study Summary

## Overview
This project demonstrates a full internal penetration test across three systems, simulating real-world attack paths from initial access to privilege escalation.

## Tools Used
- Nmap
- Wireshark
- Metasploit
- SQLMap
- DirBuster
- GTFOBins

## Skills Demonstrated
- Network reconnaissance
- Vulnerability exploitation
- Privilege escalation (Windows + Linux)
- Web application exploitation (SQLi)
- Credential analysis and abuse

## Key Outcome
All three systems were fully compromised, demonstrating chained attack paths and multiple privilege escalation techniques.

## Key Learning Outcome
Security weaknesses in authentication, configuration, and input validation can lead to full system compromise if combined in attack chains.
