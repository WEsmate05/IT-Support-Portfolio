# Network Scan (Nmap)

## Objective
Identify devices and open ports on a local network.

## Command Used
nmap -sS 192.168.1.0/24

## Findings
- Multiple active devices detected
- Common open ports:
  - 80 (HTTP)
  - 443 (HTTPS)
  - 22 (SSH)

## Outcome
Demonstrates network discovery and basic security analysis skills.
